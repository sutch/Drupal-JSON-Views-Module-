Drupal JSON Views Module
========================

What's This?
------------
This is a views backend module for supporting JSON from source.
This module's code is based on views_xml_backend.

How To Use?
-----------
1. Install this module.
2. Add a new view.
3. On the Add a new view page, choose to Show JSON.
4. Within Query settings, identify the JSON file and the Row Apath.
5. For each field, Add a new "JSON: Value" and identify the value's JSON key.
